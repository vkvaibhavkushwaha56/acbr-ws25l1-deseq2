# Start R
To start R Statistical Programming Language, type R in the command line of the R4.5WS conda evironment

```{cmd}
R
```
# Install Packages for Bioconductor
Installing `BiocManager` from CRAN (Comprehensive R Archival Network) using `install.packages()` function in the R base. Please type the following,
```{R}
install.packages("BiocManager")
```

Install `Biostrings` Bioconductor package using `BiocManager::install()` function.
```{R}
BiocManager::install("Biostrings")
